# Data Visualization – Population Distribution

This project is part of my Data Science Internship at Prodigy InfoTech.

## Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable such as age or gender in a population dataset.

## Project Description
In this project, I used Python libraries to visualize population distribution using bar charts and histograms. Data visualization helps in understanding patterns and trends within datasets.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome
The visualization clearly shows how population is distributed across different categories, helping in easier data interpretation.
